# 什么是 ArcGIS

ArcGIS 是一一个平台、统称，而非特指某个程序，包含以下三种制图及可视化的程序：

  • ArcMap 常用于制图、编辑、分析和数据管理等，一般用于 2D 对象
  • ArcGlobe 用于通过连续的全球视图实现地理数据的无缝 3D 可视化，属于可选 ArcGIS 3D Analyst 扩展模块的组成部分 。
  • ArcScene 用于实现所关注场景或区域的 3D 可视化，属于局部地区 3D 场景。也是上述扩展的组成部分

ArcMap 文档 也称为地图文档（.mxd 文件）可理解为一个容器，用于存放并显示图层等数据。（注意，存放的数据若被非 ArcMap 软件删除，将无法显示在地图文档中）

ArcMap 视图 有两种：

  • 数据视图 简单理解为在该视图下，可以对地图（数据框）进行数据处理；（数据处理）
  • 布局视图 简单理解为在该视图下，可对地图进行美化加工最终得到一幅成品地图。（制图）

数据框 简单理解为地图的另一种叫法。

地图图层（layer） 同样可理解为一个容器，用于存放并显示地理数据集，如：河流、道路等。

内容列表 用于展示当前地图文档中包含的所有内容，简单用书籍的目录理解。

页面布局 简单理解为排版设计。

目录窗口 默认位于右侧，作用是方便数据组织与管理。

主目录文件夹 即默认工作目录，当保存地图、数据集等数据时，不指定位置，系统将会存放于一个默认的文件夹及其中的默认地理数据库中。

标注 对图层中的要素做标识，是字符串。

注记

符号 简单来说，就是某要素的图形化。

# 添加工具

可以通过以下三种方法对 ArcMap 的界面添加 / 删除工具：

  • 菜单栏处右键

  • 单击菜单栏自定义 >> 工具条

  • 单击菜单栏自定义 >> 自定义模式

在自定义模式中选择『从文件添加』可添加第三方工具条。

# 恢复默认界面

在 Windows 文件资源管理器中粘贴如下路径 C:\Users\hu\AppData\Roaming\ESRI\Desktop10.2\ArcMap\Templates ，在此目录下删除 Normal.mxt 文件即可。

注 AppData 为隐藏目录，单独查找需要在 Windows 文件资源管理器中开启显示隐藏目录。

# 自定义快捷键

单击菜单栏自定义 >> 自定义模式 >> 键盘，选择类中指定的命令，在新建快捷按键中输入指定按键，单击分配即可。

# 编辑工具

导入数据 >> 加载编辑器工具条 >> 编辑器 >> 开始编辑 >> 选择编辑的要素。

右侧出现该窗口，创建要素窗口下为待操作的要素图层，需要单击选中。构造工具窗口下为待添加的要素，需要单击选中。选择完毕后即可在该图层中单击鼠标左键绘制指定要素，或使用右键中的一些选项辅助绘制指定的要素。

编辑工具的使用注意：

  • 图层不显示时是无法进行要素创建的。

构造线要素中右键的一些选项说明：

  • 长度：手动输入线段长度。

  • 绝对 XY：输入的 xy 坐标就是该地图文档的 xy 坐标

  • 增量 XY：输入的 xy 坐标是以上一个点为原点作为参考系

  • 平行、垂直：右键选中一条线，在单击这两个选项，结果就是平行或垂直于选中的那条线。

# 移动

  • 也可以直接在编辑工具条 >> 编辑器 >> 移动中使用

作用：输入 xy 坐标，将选中的要素移动到当前地图文档的该坐标上。

# 分割多部份要素

多部分要素，就是一对多的关系。一条记录对应多个要素类。

单部分要素则是一对一。

>> 多部分至单部分

或者菜单栏空白处 >> 高级编辑工具条

使用工具和高级编辑工具条的区别：

  • 工具不改变原始数据，生成一个新的仅包含分割出来的单部分要素图层；
  • 高级编辑工具条直接修改原数据。

二者新的属性表字段面积继承原始数据，需要重新使用几何计算器计算。

# 绘制中点连线

可以使用编辑工具中的中点工具

开始编辑 >> 创建线要素 >> 选择中点工具

每次鼠标绘制一条线段，工具会自动取中点，然后将各中点连线。

# 线要素裁剪与延申

开始编辑后，选中面要素，在高级编辑工具条分别选择裁剪、延伸，然后单击线要素

软件将会自动将面没有完全与面相接的线段延伸 / 裁剪

# 绘制带空洞的面要素

简单理解：面要素中有空的部分。

法一：

  • 先绘制整体
  • 再绘制空的部分
  • 选中空的部分，编辑工具条中，编辑器 >> 裁剪 >> 丢弃相交区域，缓冲距离默认，删除绘制空的部分，完成。

法二：

  • 同理绘制整体，不要双击或右击完成草图
  • 右键完成部件，然后接着绘制空的部分
  • 空的部分没绘制完就用完成部件，否则完成草图即可。

# 线要素分割面要素
